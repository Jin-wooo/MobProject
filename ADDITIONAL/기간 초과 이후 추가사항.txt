기간을 넘긴 이후의 수정사항

1. MapActivity의 하단 바 추가
 - 다른 곳에서는 하단 바가 있었으나, 제가 만든 MapActivity에만 하단 바가 없어서
   그 부분을 추가했습니다.

2. 뒤로 가기 키를 눌렀을 때 액티비티 이동 추가
 - stActivity와 MapActivity에서 Main으로 돌아올 수 있도록 개선했습니다.

3. 디버그에서의 실행 불가로 인한  APK 파일 추가
 - 카카오맵은 AVD에서는 실행이 불가능한 문제가 있었습니다
   해당 문제는 아예 카카오가 불가능하다고 못을 박아버렸기 때문에,
   이를 해결하기 위해 릴리즈된 APK 파일을 첨부합니다.

4. 위치 '요청' 기능 개선
 - 위치 요청 기능이 요청을 제대로 할 수 있도록 개선했습니다.

5. README 추가
 - README에 설명이 빠진 부분을 추가했습니다.
   구체적으로는 편의점 검색 부분의 보강 설명입니다.
